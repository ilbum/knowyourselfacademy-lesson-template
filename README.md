# Knowyourself Academy Lesson Template

This is for creating lesson documents in HTML.

## Guide

### Usage

```
sass --watch scss/custom.scss:assets/custom-bootstrap.css
```

- first → Red
- second → Orange
- third → Yellow
- fourth → Green
- fifth → Blue
- warm-dark → Pantone Warm Gray 10
