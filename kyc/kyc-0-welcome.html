Terminology

Computer

It is a device that executes instructions written in programming language. The instructions are arithmetic or logic operations.

Code

To code and program are generally interchangeable. It is to write instructions that a computer can execute.

Execute

Execute is a common computer term which means to do, perform, or carry out.

Arithmetic

Arithmetic means addition, subtraction, multiplication, and division. What is typically called math. Although math is much more than aritmetic. Computers are amazing at arithmetic, modern computers can do more than a billion arithmetic calculations a second!
